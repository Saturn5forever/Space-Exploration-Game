# **Void Space**

The repo for my game about racing spaceships around planets with fully simulated gravity.

Hello, I am Saturn and I'm making an open sourced game about racing spaceships through planet infested waters. This game includes:
- 5 Levels
- Simulated Gravity
- Fun SFX
- Highscore/Score System

Itch.io link [https://saturn5forever.itch.io/void-space-alpha](url)
